  Features:
  - Hides posts from all 50 governors, 100 senators, and 435+ representatives
  - Toggle on/off from the extension popup
  - Shows count of hidden posts
  - Works with X.com's infinite scroll
  - Logs hidden posts to the console for transparency

  To install:
  1. Open Chrome and go to chrome://extensions/
  2. Enable "Developer mode" (top right toggle)
  3. Click "Load unpacked"
  4. Select the /Users/goose/Projects/chrome-plugins/unrepresentative folder
  5. Visit x.com and posts from elected officials will be hidden

  Optional - Add icons:
  1. Open generate-icons.html in a browser
  2. Right-click each canvas and save as the indicated filename (icon16.png, icon48.png, icon128.png)
  3. Place them in the extension folder and reload the extension
